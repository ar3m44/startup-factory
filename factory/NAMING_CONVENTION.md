# Naming Convention

## Tasks
- Use sequential IDs: `TASK-0001`, `TASK-0002`, etc.
- Task files live in `factory/tasks/`.

## Branches
- Use `feat/task-0001` for task branches.

## Reports
- Use `factory/results/TASK-0001-REPORT.md`.

## Ventures
- Store ventures in `ventures/<slug>/`.
- Use lowercase kebab-case for `<slug>` (example: `ai-notes`).

## Code
- Use kebab-case for new directories and files unless the framework requires
  a different convention.
- Use PascalCase for React components.
