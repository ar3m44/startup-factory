# Results Directory

Эта директория содержит отчёты Codex о выполненных задачах.

## Структура

Каждый отчёт — это файл формата `REPORT-XXXX.md`, где:
- `XXXX` — номер задачи, совпадающий с `TASK-XXXX.md`

## Пример

```
factory/results/
  REPORT-0001.md  — Отчёт по TASK-0001
  REPORT-0002.md  — Отчёт по TASK-0002
  REPORT-0003.md  — Отчёт по TASK-0003
  ...
```

## Создание отчёта

Используй шаблон из `factory/templates/REPORT_TEMPLATE.md`.

## Содержание отчёта

Отчёт должен включать:
- Статус выполнения
- Время выполнения
- Изменённые файлы
- Ссылку на PR
- Проблемы и решения
- Скриншоты/демо (если применимо)
